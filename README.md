## 三、目录说明
### \web
> 该目录下主要存储前端页面程序  
>>css--层叠样式表  
>>imgs--所有图片  
>>news--网站新闻  
>>store_manager_page--网站普通管理员界面  
>>tom_manager_page--网站高级管理业界面  
>>tag--导航栏  
>>verification--验证相关页面  

### \src
> 该目录下主要存储后台程序
>> com.wry  
>>> dao--dao层  
>>> domain--JavaBean  
>>> JDBC  
>>> utils--Util层  
>
>> filter--过滤器  
>> servlet--control层
